## Usage

```tsx
import { Kbd, KbdGroup } from "@/components/ui/kbd"
```

```tsx
<Kbd>Ctrl</Kbd>
```

## Composition

Use the following composition to build `Kbd` and `KbdGroup`:

```text
Kbd
KbdGroup
├── Kbd
└── Kbd
```

## Examples

### Variants

Use the `variant` prop on `Kbd` for the default filled style or a lighter appearance (`variant="light"`).

<ComponentPreview name="kbd-variants" />

### Group

Use `KbdGroup` to group multiple keys with consistent spacing.

<ComponentPreview name="kbd-group" />

### Button

Put `Kbd` inside `Button` to show an accelerator hint next to the label. Use `data-icon="inline-end"` so spacing respects RTL.

<ComponentPreview name="kbd-button" />

### Tooltip

Use `Kbd` inside `TooltipContent` to show shortcuts in hover hints.

<ComponentPreview name="kbd-tooltip" />

### Input group

Show shortcuts inside `InputGroupAddon`, for example on the trailing edge of a search field.

<ComponentPreview name="kbd-input-group" />
