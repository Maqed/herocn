---
title: Introduction
description: Maqed/ui is a set of beautifully-designed, accessible components.
---

Maqed/ui ships with a **custom shadcn-style registry** that lets you pull in components on demand, wire them up with your design tokens, and evolve them as your product grows—without locking you into a black-box component library.

This particular component is **fully styled** and **heavily inspired by [HeroUI v3](https://v3.heroui.com/)**, borrowing its modern, card‑driven aesthetic and interaction patterns while still remaining completely open and customizable.

At the same time, the overall developer experience aims to offer **the same level of abstraction and flexibility as [shadcn/ui](https://ui.shadcn.com/)**: copy‑based components, local source control, and a registry-driven workflow so you can treat these building blocks as a starting point for your own design system rather than a fixed set of widgets.
